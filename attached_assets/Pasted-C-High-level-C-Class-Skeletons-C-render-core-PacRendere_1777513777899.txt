};

These classes are intentionally minimal and focused for M2.5 MVP. They prioritize:

Fast simulation → render sync
Easy glTF import
Clean atmospheric defaults

Editor Changes
Add a toggle button in the top bar: 2D Debug ↔ 3D Atmospheric
When in 3D mode, replace the current orthographic canvas with the new 3D viewport
Send simulation deltas from your existing timeline/replay system to the renderer
Use the same Outliner + Entity Inspector (just show additional render properties when in 3D mode)

Data Flow
Simulation tick finishes → minimal delta (entity transforms, active agents, etc.)
Delta sent to PacRenderer::UpdateSimulationState()
Renderer updates only dirty RenderProxy objects
Frame rendered with atmospheric effects applied
